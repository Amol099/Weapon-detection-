# Real-Time Gun Detection System using OpenCV

A Computer Vision security application that detects firearms in real-time using a webcam and a Haar Cascade classifier. The system processes live video frames, identifies potential weapons, and highlights detected objects with bounding boxes for surveillance and security monitoring.

🚀 Overview

This project is a real-time firearm detection system developed using Python and OpenCV. It utilizes a pre-trained Haar Cascade classifier to detect guns from live webcam footage.

The application continuously captures video frames, converts them into grayscale for efficient processing, identifies potential firearms, and draws bounding boxes around detected objects. This project demonstrates the fundamentals of computer vision, object detection, and real-time video processing for security and surveillance applications.
